## What I did for this project:
1. Used [**Hugo**](https://gohugo.io/) to build my blog
2. Deployed on [**AWS S3**](https://aws.amazon.com/s3/)
3. Used [**AWS CloudFront**](https://aws.amazon.com/cloudfront/) as CDN
4. Used [**AWS Certificate Manager**](https://aws.amazon.com/certificate-manager/) to generate SSL certificate
5. Bought a domain name on [**GoDaddy**](https://www.godaddy.com/)
6. Used [**Cloudflare**](https://www.cloudflare.com/) as DNS

## What I learned:
1. Hands-on experience with AWS services (S3, CloudFront, Certificate Manager)
2. Reviewed fundamental concepts: DNS, CDN, SSL/TLS
3. Learned about popular service providers: GoDaddy, AWS, Cloudflare
